# Naming Conventions

Правила именования, используемые в проекте.

## Компоненты
Используем **PascalCase**.

**Примеры:**
- `HeaderMenu.astro`
- `UserCard.jsx`
- `LoginForm.tsx`

## Константы
Используем **UPPER_SNAKE_CASE**.

**Примеры:**
- `API_URL`
- `DEFAULT_TIMEOUT`
- `MAX_RETRY_COUNT`

## Папки и файлы
Используем **kebab-case**.

**Примеры папок:**
- `user-profile/`
- `main-layout/`
- `shared-components/`

**Примеры файлов:**
- `scroll-animation.js`
- `error-page.astro`
- `form-utils.ts`

## CSS-классы
Используем **kebab-case**.

**Примеры:**
- `.main-container`
- `.header-wrapper`
- `.button-primary`
- `.section-title`
