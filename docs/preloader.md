# Preloader.astro
 
Мок-компонент загрузочного экрана. Используется только для демонстрации заказчику — с реальной загрузкой не связан.
 
## Что делает
 
Показывает полноэкранный оверлей с анимированным счётчиком процентов и прогресс-баром. Цифры проходят через фиксированные значения по таймеру:
 
```
0% → 54% → 67% → 98%
```
 
Останавливается на 98% — намеренно, чтобы показать что дальше должна появиться страница.
 
## Как поменять шаги
 
В скрипте компонента отредактировать значения в массиве `steps` :
 
```ts
const steps: Step[] = [
  { value: 0,  delay: 0    },
  { value: 54, delay: 800  },
  { value: 67, delay: 2000 },
  { value: 98, delay: 3200 },
];
```
 
`delay` — время в миллисекундах от старта до начала этого шага.
 
## Если нужен fade out
 
Добавь в конец `runSteps`, после последнего шага:
 
```ts
const el = document.getElementById('preloader') as HTMLElement;
el.style.transition = 'opacity 0.6s ease';
el.style.opacity = '0';
setTimeout(() => el.remove(), 600);
```
